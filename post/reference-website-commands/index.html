Website Reference

Published September 24, 2021

This page summarises the commands used to build this site to save time on looking up as this writer posts too infrequently and forgets.

Website creating commands

To create new post:
hugo new post/my-first-post.md
and edit new markdown file after.

To start hugo server with drafts enabled:
hugo server -D
and navigate to http://localhost:1313/.

To build static pages:
hugo -D
then copy to repository and push.
